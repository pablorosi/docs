Welcome to the **Enterprise Homelab** documentation. This environment is the foundation for **NetDevOps** experiments, Cisco networking labs, and Linux systems administration.

<Aside type="caution" title="Evolution in Progress">
The lab is continuously evolving. Infrastructure docs (sections 1–7) are updated as hardware and services are provisioned.
</Aside>

## Architecture

<ThemeImage
  lightSrc="/network-topology-light.svg"
  darkSrc="/network-topology.svg"
  alt="Hybrid Lab Network Topology"
/>

*Figure 1: Physical network architecture with collapsed core design.*

A **hybrid infrastructure** combining physical Cisco hardware with a **Proxmox** cluster: physical switches for real Layer 2/3 behavior, virtual machines for emulation and services.

## Current Status

<Steps>
1. **Architecture:** HP t740 compute node, Catalyst 3560-CX core, 2960 access — documented in [Hardware Specs](/enterprise-homelab/1-architecture/hardware-specs).
2. **Hypervisor:** Proxmox VE on the t740 with nested virtualization for EVE-NG.
3. **Networking:** VLANs, trunking on Cisco switches; Tailscale mesh for remote access.
4. **Emulation:** EVE-NG bridging virtual IOS nodes to physical switch ports.
5. **Observability:** Prometheus and Grafana — **planned (post-CCNA)**.
6. **Phase 1 backlog:** [IPAM](/enterprise-homelab/ipam/) for address management; Ansible for VLAN/switch automation.
7. **Phase 2 backlog:** [Autonomous SRE Agent](/local-sre-platform/) — planned architecture, requires observability baseline first.
</Steps>
