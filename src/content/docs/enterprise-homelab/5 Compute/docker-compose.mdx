## Role in the Homelab

Compose files define reproducible multi-service stacks — monitoring, databases, and automation tools — deployed on the Proxmox compute layer.

## Planned Stacks

Docker Compose is the deployment model for the **Local SRE Platform**: Prometheus, Grafana, alerting, and supporting services as a unified stack.

## Current Status

Individual compose files and the central repository layout will be documented here as stacks are deployed on the homelab.

See **[Docker Engine](/enterprise-homelab/5-compute/docker-engine)** for host setup and **[Observability](/enterprise-homelab/6-observability/prometheus)** for the monitoring roadmap.
