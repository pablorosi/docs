## Image Types

EVE-NG supports multiple Cisco image formats for lab topologies:

| Type | Use Case |
| :--- | :--- |
| **Cisco IOL** | Lightweight IOS on Linux — routers and switches |
| **QEMU** | Full IOS/IOS-XE images — higher fidelity, more resource-intensive |

The EVE-NG VM is sized for both IOL and QEMU workloads — see **[EVE-NG Deployment](/enterprise-homelab/4-emulation/eve-ng-deployment)**.

## Loading Images

Step-by-step procedures for uploading and licensing IOU/QEMU images in EVE-NG will be documented here. Current lab uses **Cisco IOL** nodes for routing and switching emulation.
