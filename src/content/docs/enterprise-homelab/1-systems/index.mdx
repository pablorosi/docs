This section documents the compute and storage infrastructure that powers my labs. My philosophy is to use high-efficiency, small-form-factor hardware to build a scalable and quiet environment.

<Aside type="note" title="Philosophy">
I prioritize the **Performance-per-Watt** ratio, which led me to select the HP t740 platform as my primary hypervisor node.
</Aside>

## Core Software Stack

*   **Hypervisor:** Proxmox VE (Debian-based)
*   **Virtualization:** KVM / QEMU
*   **Containerization:** LXC and Docker
*   **Access & Security:** SSH, Tailscale, and RSA-key authentication
