## Role in the Homelab

**Prometheus** will collect time-series metrics from network devices, Proxmox hosts, and containerized services. It is the metrics backbone of the planned **Autonomous SRE Agent**.

## Planned Scope

* Node and VM metrics from the Proxmox hypervisor.
* SNMP or exporter-based metrics from Cisco switches.
* Container and application metrics from Docker workloads.

## Scrape Targets
