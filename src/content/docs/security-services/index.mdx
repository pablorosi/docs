Welcome to the **Security & Core Services** documentation.

Instead of reinventing the wheel for every new server or application, both the Homelab and Cloud Infrastructure projects rely on a centralized, standardized networking and security layer. 

## 🔐 Security Philosophy: Zero Trust

All internal administration panels and management interfaces (such as EVE-NG or Nginx Proxy Manager) adhere strictly to a **Zero Trust Network Access (ZTNA)** model. 

Our core architectural principles are:
1. **No Public Ports:** Firewalls are configured to drop all inbound public traffic to management interfaces by default.
2. **Private Overlay Routing:** Access is strictly routed through an encrypted Tailscale mesh network (`100.x.x.x`).
3. **Valid Cryptography:** All internal endpoints use fully trusted Let's Encrypt SSL certificates validated via **DNS-01 challenges**, ensuring data integrity without exposing the servers to HTTP-01 internet validation.
