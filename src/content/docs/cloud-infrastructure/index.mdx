## Infrastructure Pillars

This section documents the backend environment that hosts [pablorosi.dev](https://pablorosi.dev) and [docs.pablorosi.dev](https://docs.pablorosi.dev).

### 1. Secure Cloud Routing

This layer handles all incoming traffic, edge security, and internal service routing. It details the **Defense in Depth** strategy, including the Hetzner Cloud Firewall, Nginx Reverse Proxy configurations, and the Zero Trust Tailscale overlay network used to secure administrative access.

### 2. Troubleshooting

Operational guides for diagnosing and resolving common issues in the production environment.
